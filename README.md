# Jinja Senior Secondary School Platform

**"Education for Service"**

A comprehensive school management platform with 50 role-based dashboards covering all aspects of school operations.

---

## Technology Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML5, CSS3, JavaScript (Vanilla) |
| Backend | Python + FastAPI |
| Database | SQLite |
| Real-time | WebSockets |
| Authentication | JWT |
